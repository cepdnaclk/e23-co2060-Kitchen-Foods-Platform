# Kitchen Foods Super Admin (Isolated App)

This is a standalone React + TypeScript admin dashboard application for Kitchen Foods.

## Stack

- React (functional components + hooks)
- React Router
- Tailwind CSS
- Axios
- React Context API (auth state)
- React Icons

## Run Admin App

```bash
cd admin
npm install
npm run dev
```

Default local URL:

`http://localhost:5173/admin/login`

## Build

```bash
npm run build
```

## API Base

Admin API calls are prepared for:

`http://localhost:5000/api/admin`
