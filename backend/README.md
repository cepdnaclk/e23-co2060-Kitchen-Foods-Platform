# Backend - Kitchen Foods Platform

## API Documentation

- Primary Markdown reference: `docs/API_REFERENCE.md`
- Interactive Swagger UI: `http://localhost:8000/api-docs`
- Raw OpenAPI contract: `docs/openapi.json` (served at `/api-docs/openapi.json`)

## Run Backend

```bash
npm install
npm run dev
```

If required for local services:

```bash
docker compose up -d
```
